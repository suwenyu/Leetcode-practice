## 326. Power of Three

Given an integer, write a function to determine if it is a power of three.

### Example
```
Input: 27
Output: true

Input: 0
Output: false

Input: 9
Output: true
```

### My Code
```python
threepowList = [pow(3,i) for i in range(0,20)]
if n in threepowList:
    return True
else:
    return False
```

### Others Solution
```python
return n > 0 and 3 ** round(math.log(n, 3)) == n
```
